# Mahmoud Mohamed Portfolio

A personal portfolio website built to showcase my frontend development work, technical skills, experience, and selected projects.

## Live Demo

https://mahmoud-mohamed-portfolio.vercel.app

## Features

* Responsive landing page
* About section
* Work history
* Featured projects
* Technical skills
* Contact links
* CV download

## Tech Stack

* Angular
* TypeScript
* HTML5
* SCSS
* Bootstrap
* Vercel

## Sections

* Hero
* About Me
* Services
* Work History
* Featured Projects
* Technical Skills
* Contact

## Run Locally

```bash
git clone repository-link
cd portfolio
npm install
ng serve
```
